# Deep Learning Image & NLP Classification Project

## Overview

This project explores deep learning techniques for both image classification and natural language processing (NLP) tasks using TensorFlow and Keras.

Note : (Dataset available upon request)

The research compares multiple deep learning architectures, including:

* CNN
* MobileNetV2
* ResNet50
* RNN
* LSTM
* BERT

The project evaluates model performance, transfer learning strategies, fine-tuning approaches, and interpretability techniques across image and text classification datasets.

---

## Features

* Image classification using CNN and transfer learning
* NLP-based news topic classification
* Deep learning model comparison
* Transfer learning and fine-tuning
* Data preprocessing and augmentation
* Model evaluation and accuracy analysis
* Interpretability analysis using t-SNE and LIME

---

## Results

* Achieved up to 93% validation accuracy using BERT for NLP classification
* Achieved up to 90% validation accuracy using MobileNetV2 for image classification
* Compared performance between lightweight and heavyweight deep learning models
